# Contributing to CV Tailor NG

Thank you for your interest in contributing! By contributing, you agree to follow the guidelines below.

## How to Contribute

1. Fork the repository
2. Create a new branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Run the project and tests to ensure nothing breaks
5. Commit your changes: `git commit -m "Add feature"`
6. Push to your branch: `git push origin feature/my-feature`
7. Open a Pull Request

## Code Guidelines

- Use TypeScript for all new components
- Tailwind CSS for styling
- Keep code clean, readable, and maintainable
- Follow the Paper Design system (SKILL.md)

## Reporting Issues

- Use GitHub Issues to report bugs or request features
- Be descriptive and include screenshots if possible

# Changelog

All notable changes to CV Tailor NG are documented here.

## [1.0.0] — 2025

### Added
- **Dashboard** — 4-step CV upload, sector selection, AI tailoring, result pane
- **Resume Builder** — one-page two-column modern resume generator
- **Cover Letter Generator** — formal Nigerian cover letter (Dear Sir/Ma · Yours faithfully)
- **Email Templates** — 5 Nigerian professional email templates with send functionality
- **Bio & Profile Sheet** — LinkedIn, Twitter/X, GitHub bios + 100-word short bio
- **Templates Gallery** — 6 Nigerian sector-specific CV templates
  - Banking & Finance (ICAN, CBN, AML/CFT language)
  - Oil & Gas (HSE, NUPRC, NCDMB, NAPIMS)
  - Technology / Fintech (NIBSS, CBN sandbox)
  - NGO / Development (M&E, USAID, DFID, UN)
  - Civil Service / Public Sector (MDAs, IPPIS, GIFMIS)
  - FMCG / Sales (route-to-market, Nielsen data)
- **Download Formats** — PDF, DOCX, HTML, Plain Text for all document types
- **Direct Email Send** — SMTP integration with CV attached as PDF
- **CV File Parsing** — PDF, DOCX, TXT upload support
- **Nigerian Standards**
  - NYSC Status field (Discharged / Exempted / Ongoing)
  - State of Origin field
  - Professional body references (ICAN, CIPM, COREN, NSE, NIM, NIMN, ACCA, PMP)
  - Sector-specific vocabulary (18+ sectors)
  - Correct Nigerian salutations and sign-offs
  - Naira (₦) formatting
  - Jobberman / LinkedIn advert detection
